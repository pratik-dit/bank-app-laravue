# Bank App

###### Setup Project ######

- `composer install`
- create new `.env` file and copy content from `.env.example`
- update database credential into `.env` file
- `php artisan key:generate`
- `php artisan migrate`
- `npm install`
- `npm run dev`
- `php artisan serve`

#####  Test Case Run #####
- `./vendor/bin/phpunit tests/Feature/AuthTest.php`
